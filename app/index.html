Backbone, Raphael and Transformable Demo by Will Demaine

This is a live demo which combines Backbone, RaphaelJS and Raphael Free Transform. It uses my 'backbone.raphael.transformable' which provides a way to link backbone with RaphaelJS. The source code for this demo is available on GitHub and below. This demo uses Backbone.localStorage to persist the shapes. Try changing them and reloading the page to test that it's working.
